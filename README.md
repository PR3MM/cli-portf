
# CLI Portfolio

Interactive CLI-style portfolio built with React, Vite, and Tailwind CSS. It exposes a small in-app terminal where visitors can explore files and view project details.

## Features

- Terminal-like UI with commands: `ls`, `cd`, `cat`, `pwd`, `clear`, `help`
- Self-contained filesystem with about, skills, projects, and blog entries
- Built with Vite for fast dev experience and Tailwind for styling
